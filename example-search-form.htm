Example including search form



This page shows you how to include Search Form in your website pages, to use it with SSEP Search Engine Script.
- After you have indexed pages of your web site (see the "ssep/readme.html" page), just add this html code in website pages: 
<form action="ssep/index.php" method="post">
  <input type="text" name="sr" maxlength="45" />
  <input type="submit" value="Search" />
</form>
The address from "action" must open the "ssep/index.php" file.
